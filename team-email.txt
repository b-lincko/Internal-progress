📋 WHAT THIS IS USEFUL FOR

As we work toward CMMC Level 2 compliance, this tool helps us:

✅ Track 108 CMMC controls across 14 security domains
✅ Monitor deadlines, schedules, and milestones
✅ Upload and manage compliance evidence and documents
✅ Track IT assets and network inventory
✅ Plan of Action & Milestones (POA&M) management
✅ Team collaboration via built-in chat (Global + Private)
✅ Project tracking with tasks and progress
✅ Real-time notifications for updates
✅ Generate SSP (System Security Plan) PDF exports

───────────────────────────────────────

🎯 KEY FEATURES

🔐 ROLE-BASED ACCESS
• Admin — Full access, user management, admin panel
• Manager — Can edit controls, create deadlines, upload docs, create projects
• Viewer — Read-only access + chat + view assigned items

🛡️ CMMC CONTROLS
• All 108 CMMC 2.0 controls pre-loaded
• Status tracking: Not Started → In Progress → Implemented
• Link evidence and create POA&Ms per control

📅 SCHEDULES & DEADLINES
• Create compliance deadlines
• Assign to team members
• Priority levels: Low, Medium, High, Critical
• Overdue highlighting

📁 DOCUMENT MANAGER
• Folder-based file explorer
• Upload compliance evidence (PDFs, images, etc.)
• Access control per user (Read/Write/Admin)
• Audit trail — see who did what and when

💬 TEAM CHAT
• Global chat room for team announcements
• Private messaging between users
• File sharing in chat
• Voice messages and file uploads

📊 PROJECT TRACKER
• Create compliance projects
• Add tasks and assign members
• Track progress

💻 ASSET INVENTORY
• Track IT assets: servers, workstations, laptops, network devices
• Status tracking: Active, Inactive, Decommissioned, Maintenance
• Criticality levels

🔔 NOTIFICATIONS
• Automatic notifications for schedules, documents, projects
• Unread badge in sidebar
• Real-time updates

🌓 PROFESSIONAL UI
• Dark theme with glass card design
• Mobile-friendly responsive layout
